## 🛠️ Key Categories & Top Picks (关键分类与顶级推荐)

### 1. Agent 核心框架 (Core Frameworks)
*用于构建 Agent 逻辑、状态管理和工具调用。*
- **AutoGPT / BabyAGI (Next-Gen)**: 经典的自主任务分解架构，2026 年版重点在于长短期记忆的融合。
- **LangGraph / CrewAI**: 专注于多 Agent 协作 (Multi-Agent Orchestration) 的首选，支持复杂的有向无环图 (DAG) 工作流。
- **OpenAI Assistants API (v2+)**: 极简集成，内置强力状态管理与文件检索。

### 2. 编码与技术 Agent (Coding & Technical Agents)
*专注于代码库维护、自动化迁移和软件工程。*
- **Claude Code**: 目前最强的代码库规模迁移与编辑工具。
- **Devin / OpenDevin**: 具备完全自主操作环境的 AI 软件工程师。
- **Cursor (Agent Mode)**: 将 IDE 与 Agentic Workflow 结合的行业基准。

### 3. 多模态与创意 Agent (Multimodal & Creative Agents)
*处理视觉、音频和复杂创意流。*
- **Voice-Agent Frameworks**: 支持超低延迟实时语音交互的协议。
- **Gen-AI Image Orchestrators**: 能够根据需求自动迭代 Prompt 并生成一致性视觉方案的 Agent。

### 4. 企业级与行业 Agent (Enterprise & Vertical Agents)
*针对法律、医疗、金融等强合规场景。*
- **Legal Agent Benchmark Top-Performers**: 专注于复杂合同分析和法律检索的专用 Agent。
- **Enterprise Knowledge Base Agents**: 能够处理千万级 Token 上下文并保证零幻觉的知识库 Agent。
