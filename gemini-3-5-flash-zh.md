# Gemini 3.5 Flash 能力规格书 (AI-Ready Version)

## 1. 定位与核心能力 (Core Positioning)
- **核心定位**：面向 Agentic Workflow 的前沿智能模型 (Action-Oriented Frontier Model)。
- **关键目标**：在维持 Flash 系列极高响应速度的同时，提供足以驱动复杂、长程 (Long-horizon) 自主任务的推理能力。
- **核心特性**：
  - **Agent 基因**：原生优化用于“计划 $\rightarrow$ 执行 $\rightarrow$ 迭代”的闭环任务。
  - **极致速度**：输出 Token 速度比其他前沿模型快 **4 倍**。
  - **规模化协同**：配合 Antigravity 框架，支持大规模部署协作子代理 (Subagents)。
  - **多模态增强**：能生成更丰富、更具交互性的 Web UI 与图形界面。

## 2. 核心能力矩阵 (Performance Matrix)

### 🛠️ 编程与 Agent 执行 (Coding & Agentic Execution)
- **代码执行力**：Terminal-Bench 2.1 准确率 **76.2%**。
- **综合 Agent 能力**：MCP Atlas 得分 **83.6%**；GDPval-AA 达到 **1656 Elo**。
- **多模态推理**：CharXiv Reasoning 表现强劲，得分 **84.2%**。

### ⚡ 效率与成本 (Efficiency & Cost)
- **延迟表现**：继承 Flash 系列特性，极低首字延迟 (TTFT)。
- **成本优化**：在执行长程任务时，成本通常低于其他前沿模型的 **50%**。
- **通量**：针对大规模自动化流水线优化，支持高并发子代理调用。

### 🤖 典型应用场景 (Real-world Impact)
- **端到端自动化**：将杂乱的 legacy 代码库迁移至 Next.js；自动化重命名并分类非结构化资产。
- **企业级集成**：
  - **Shopify**：并行运行子代理进行全球商户增长预测。
  - **Macquarie Bank**：低延迟解析 100+ 页复杂文档并给出可靠建议。
  - **Salesforce (Agentforce)**：部署多子代理执行复杂的多轮工具调用。
- **个人助理**：驱动 **Gemini Spark** (24/7 运行的个人 AI Agent)。

## 4. AI 选型建议 (Selection Guide)
- **优先调用场景**：
  - 需要【极快响应速度】且【任务链路长】的 Agent 自动化。
  - 需要【低成本大规模部署】子代理的复杂工程。
  - 需要【多模态 $\rightarrow$ 交互式 UI】的快速原型生成。
- **对比 GPT-5.5 / Claude 4.8**：在“推理深度”上可能略逊于两者之最，但在“执行速度”和“部署成本”上具有压倒性优势。是构建高频、低延迟 Agent 链路的首选。
